# Thoughtful Choice

A clean, responsive, single-page guide that helps individuals understand their options for end-of-life planning, including ceremonies, disposition methods, and memorialization.

## Features

- Clear, compassionate guidance for funeral planning
- Ceremony options: Home Funeral, Funeral Service, Memorial Service, Graveside Service, Burial at Sea
- Disposition options: Traditional Burial, Green/Natural Burial, Cremation, Alkaline Hydrolysis, Natural Organic Reduction
- Memorialization ideas: merchandise-based and service-based options
- Helpful checklist for final considerations
- Fully responsive design

## Project Structure

- `index.html` - Main website page
- `assets/styles.css` - Styling for layout, typography, and responsive behavior

## Running Locally

Open `index.html` in any modern browser, or run a local server on port 5000.

## Customization

- Adjust colors or typography in `assets/styles.css` under the `:root` variables
- All text content lives in `index.html`
